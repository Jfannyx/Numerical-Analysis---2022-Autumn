To run the program:
Command line run 'python main.py', output four project required results

Main functions: 
zeta(x, precision)
solveEquation(x, a, precision)
integral(x, precision)
solveODE(x0, y0, x, precision) 
# x0,y0 are the starting coordinates
# precision is inputed as m in the form of 0.5 * 10 ** (-m)

Test: 
Modify the function parameters commented out in the main.py file to test the results with different parameters


Package used: 
gmpy2-1.11.1
Decimal
math
